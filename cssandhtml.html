Macbook Keyboard Shortcuts

Web Browsers

Command Shortcut
⌘T Opens a new tab.
⌘W Closes the tab you're currently on.
⌘C Copies the selected text
⌘V Pastes the text in clipboard.
⌘X Deletes the selected text and copies it.
⌘Z Undo.
⌘Y Redo.
⌘B Makes the selected text bold.
⌘I Makes the selected text italic.
⌘U Makes the selected text underlined.
